The Binomial Probability Distribution

Definition: Binomial Experiment

Definition 1

A binomial experiment possesses the following properties:

  1. The experiment consists of a fixed number, (n), of identical trials.
  2. Each trial results in one of two outcomes: success, (S), or failure, (F).
  3. The probability of success on a single trial is equal to some value (p) and remains the same from trial to trial. The probability of a failure is equal to (q = (1 - p)).
  4. The trials are independent.
  5. The random variable of interest is (Y), the number of successes observed during the (n) trials.

Example 1: Radar Detection System

Example 1

An early-warning detection system for aircraft consists of four identical radar units operating independently of one another. Suppose that each has a probability of 0.95 of detecting an intruding aircraft. When an intruding aircraft enters the scene, the random variable of interest is (Y), the number of radar units that do not detect the plane. Is this a binomial experiment?

Solution

Let‚Äôs verify each property:

  • ‚úÖ The experiment involves four identical trials (four radar units).
  • ‚úÖ Each trial results in one of two outcomes:
    • (S) denotes that the aircraft was not detected
    • (F) denotes that it was detected
  • ‚úÖ (p = P(S) = P() = 0.05)
  • ‚úÖ The trials are independent because the units operate independently.
  • ‚úÖ The random variable of interest is (Y), the number of successes in four trials.

Conclusion: Yes, this is a binomial experiment!

Example 2: Voter Preference

Example 2

Suppose that 40% of a large population of registered voters favor candidate Jones. A random sample of (n = 10) voters will be selected, and (Y), the number favoring Jones, is to be observed. Does this experiment meet the requirements of a binomial experiment?

Remark 1

In general, if the population is large and the sample size is relatively small, the conditional probability of success on a later trial given the number of successes on the previous trials will stay approximately the same regardless of the outcomes on previous trials.

Key principle: For sampling without replacement, if (n < 0.05N) (sample size is less than 5% of population), we can approximate with binomial distribution.

Example 3: Defective Fuses

Example 3

Suppose that a lot of 5000 electrical fuses contains 5% defectives. If a sample of 5 fuses is tested, find the probability of observing at least one defective.

Solution

Let (Y) = number of defectives in the sample. Then (Y (n=5, p=0.05)).

[ P() = 1 - P(Y = 0) ]

[ P(Y = 0) = (0.05)0(0.95)5 = 1 (0.95)^5 = 0.774 ]

Therefore: [ P(Y ) = 1 - 0.774 = 0.226 ]

Interpretation: There is a 22.6% chance of seeing at least one defective, even though the sample is quite small.

Example 4: Medical Treatment

Example 4

Experience has shown that 30% of all persons afflicted by a certain illness recover. A drug company has developed a new medication. Ten people with the illness were selected at random and received the medication; nine recovered shortly thereafter. Suppose that the medication was absolutely worthless. What is the probability that at least nine of ten receiving the medication will recover?

Solution Setup

Let (Y) denote the number of people who recover. If the medication is worthless, the probability that a single ill person will recover is (p = 0.3).

Then the number of trials is (n = 10) and we need to find (P(Y )).

Example 4: Solution

The probability of exactly nine recoveries is:

[ P(Y = 9) = (0.3)^9 (0.7)^1 = 10 (0.3)^9 = 0.000138 ]

Similarly, the probability of exactly ten recoveries is:

[ P(Y = 10) = (0.3)^{10} (0.7)^0 = 1 (0.3)^{10} = 0.000006 ]

Therefore:

[ P(Y ) = P(Y = 9) + P(Y = 10) = 0.000138 + 0.000006 = 0.000144 ]

Interpretation

If the medication is ineffective, the probability of observing at least nine recoveries is extremely small (0.0144%). This provides strong evidence that the medication is actually effective!

Mean and Variance of Binomial Distribution

Key Theorem

Theorem 1

Let (Y) be a binomial random variable based on (n) trials and success probability (p). Then:

[ = E(Y) = np ]

[ ^2 = V(Y) = npq = np(1-p) ]

where (q = 1 - p).

Intuition

  • Expected value: If we repeat the experiment many times with (n) trials each, on average we expect (np) successes per experiment.
  • Variance: Measures the spread of the distribution. Maximum variance occurs when (p = 0.5).
